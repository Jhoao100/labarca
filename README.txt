LA BARCA RESTAURANTE - WEB HTML/CSS/JS

Archivos:
- index.html: estructura completa.
- styles.css: diseño responsive, elegante y animado.
- script.js: menú móvil, animaciones, filtro de carta, tilt, draggable y cambio básico de idioma.
- assets/: fotos optimizadas y video.

Para abrir:
1. Descomprime la carpeta.
2. Abre index.html en Chrome.
3. Para subirlo a hosting, sube TODO el contenido de esta carpeta a public_html o www.

Cambios importantes antes de publicar:
- Reemplazar enlace de Google Maps por el exacto del restaurante.
- Confirmar horario real.
- Agregar platos de comida si ya tienen carta oficial.
- Cambiar dominio cuando lo tengas.
